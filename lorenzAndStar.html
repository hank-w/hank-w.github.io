Lorenz System Visualized Through Code

  • Lorenz & A*

Lorenz and A* Search are two mathematical models I learned about in my courses at University of Waterloo. I learned about the Lorenz System in my Ordinary Differential Equations class and I learned about A* Search in my data structures and algorithims class. I thought the derivation for the two models, along with their real life implications, were very cool, so I decided to try and replicate the mathematical expresssions through code and visualize it. I then spent an unhealthy amount of time reading research papers and learning the algorithims, pushing the limits of TypeScript beyond resonable boundaries. Finishing my whole program took an entire month of work, time I spent surviving off instant noodles and photosynthesis. If you would like to see my code, I've linked it through 3 buttons at the bottom of this page, so its easier to read without falling asleep, based on my experience reading research papers 😄

You can try my program yourself by clicking on the button above. Similar to the randomness of a Lorenz System, the visualized algorithim is randomized! Try refreshing if you would like to see both 🙂. Also, I have added a feature that lets you interact with the Lorenz System. Try clicking on the screen when the program is running to see what happens 🙂
